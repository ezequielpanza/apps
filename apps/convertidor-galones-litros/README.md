# Convertidor Galones ↔ Litros

Aplicación web para convertir galones a litros y litros a galones en tiempo real.

## Funciones

- Conversión bidireccional instantánea.
- Galón estadounidense (US) por defecto.
- Opción de galón imperial (UK).
- Diseño responsive para celular y escritorio.
- Sin almacenamiento de datos ni APIs externas.

## Tecnología

HTML, CSS y JavaScript sin dependencias.

## Desarrollo local

Abrir `index.html` en un navegador o servir esta carpeta con cualquier servidor HTTP estático.

## Deploy

- Carpeta: `apps/convertidor-galones-litros`
- Carpeta de salida: la misma carpeta
- Proyecto Cloudflare Pages: `convertidor-galones-litros`
- URL esperada: `https://convertidor-galones-litros.pages.dev`
