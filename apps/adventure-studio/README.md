# Adventure Studio

Mockup navegable de un estudio web para crear, probar y jugar aventuras gráficas point & click.

## Tecnología

- HTML
- CSS
- JavaScript

## Desarrollo local

```bash
python -m http.server 8000 --directory apps/adventure-studio
```

## Deploy

- Proyecto Cloudflare Pages: `adventure-studio`
- Carpeta publicada: `apps/adventure-studio`
- URL esperada: `https://adventure-studio.pages.dev`

## Estado

Primera versión visual. Incluye entorno principal, navegador de proyecto, room editor, inspector, browser de assets y playtest integrado.
