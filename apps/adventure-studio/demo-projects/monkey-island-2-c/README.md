# Monkey Island 2 C

Proyecto demo privado para desarrollar Adventure Studio contra una referencia de complejidad real.

## Aislamiento

Todo el contenido específico del demo vive dentro de esta carpeta.

Para removerlo al finalizar el desarrollo:

1. eliminar `demo-projects/monkey-island-2-c/`;
2. cambiar `startup-project.js` para no apuntar a este proyecto.

El motor y los editores no deben contener lógica especial para este demo.

## Estado inicial

- Project: `Monkey Island 2 C`
- Game: `Monkey Island 2 C`
- Game Resolution: `320 × 200`
- Room abierta por defecto: `Room 007`
- Background inicial: `Default`

El demo usa el mismo modelo multi-Game que cualquier otro Project de Adventure Studio.
